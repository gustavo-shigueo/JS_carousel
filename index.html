Defaults

        
Simply calling

const carousel = new Carousel('.carousel-container')

in your <script> tag will initiate a carousel with
the following default values:
{
  timer: 5000,
  usesBubbles: true,
  usesArrows: true
}

Manual + arrow + bubble

        
<head>
  <!-- Importing the Carousel class -->
  <script src="./js/Carousel.js">
  <link rel="stylesheet" href="./css/style.min.css">
</head>
<body>
  <div class="carousel-container">
    <div class="carousel-wrapper">
      
      <!-- Each div.carousel-item will be one slide -->
      <div class="carousel-item">
        <!-- Content -->
      </div>

    </div>
  </div>

  <script defer>
    const carousel = new Carousel(
      '.carousel-container',
      { timer: false }
    )
  </script>
</body>
